🚀 MERN Stack Projects

This repository contains multiple projects built using the MERN stack (MongoDB, Express, React, Node.js).

---

Each folder contains a separate MERN application with its own frontend and backend.

---

🛠️ Technologies Used

* MongoDB
* Express.js
* React.js
* Node.js

---

⚙️ How to Run a Project

1. Go to the project folder

```bash
cd project-name
```

2. Setup Backend

```bash
cd backend
npm install
npm start
```

3. Setup Frontend (open new terminal)

```bash
cd frontend
npm install
npm start
```

---

📌 Notes

* Make sure MongoDB is running locally or use MongoDB Atlas
* Update `.env` files with your configuration
* Backend usually runs on `http://localhost:3001`
* Frontend usually runs on `http://localhost:3000`
